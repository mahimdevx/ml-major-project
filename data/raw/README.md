# Shakespeare Dataset

Place the structured Shakespeare dataset files here.

Required plays:
- Hamlet
- Macbeth
- Romeo and Juliet

Recommended format:

```json
{
  "play": "Macbeth",
  "act": "Act 1",
  "scene": "Scene 3",
  "speaker": "MACBETH",
  "text": "So foul and fair a day I have not seen."
}
```

This folder stores raw source files before preprocessing.
